## 📂 ফাইল স্ট্রাকচার

```
src/
├── app/
│   ├── page.tsx          ✅ মূল পেজ (এডিটর + প্রিভিউ)
│   ├── globals.css       ✅ স্টাইল
│   └── layout.tsx
├── components/
│   └── PDFDocument.tsx   ✅ PDF রেন্ডারিং কম্পোনেন্ট
└── utils/
    └── pdf.ts            ✅ ফন্ট রেজিস্ট্রেশন + PDF ইউটিলিটি

public/
└── fonts/                ✅ ১৪টি বাংলা ফন্ট ফাইল
```

## 🎯 পরীক্ষা করুন

1. অ্যাপ চালান: `npm run dev`
2. এডিটরে যুক্তাক্ষর লিখুন: ক্ষ, জ্ঞ, ত্র
3. ফন্ট সিলেক্ট করুন
4. "Download as PDF" ক্লিক করুন
5. PDF খুলুন এবং যুক্তাক্ষর চেক করুন

## 📌 গুরুত্বপূর্ণ নোট

- সব ফন্ট `/public/fonts/` ফোল্ডারে আছে
- যুক্তাক্ষর সঠিকভাবে রেন্ডার হবে
- কোনো সার্ভার API প্রয়োজন নেই (সব ক্লায়েন্ট-সাইড)
